Summary
--------
HOPE.ino is Ardunio program suitable for most Ardunios. It was created and tested
on a USBDroid Ardunio (from Jaycar Electronics).

The purpose of this software is to communicate with all the sensors and to:-
- Save this information to the microSD card 
- Send information back to the ground station via the RFD900 modem.



Installation Instructions
---------
1. Install Arduino Software

2. Copy the x-Modem libraries to appropriate area

3. Load HOPE.ino into Arduino

4. Plug in your arduino

5. Load HOPE.ino into the Arduino
